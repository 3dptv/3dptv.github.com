"Batteries included" is outdated - use the Github option

Submitted by alexl on Thu, 2011-09-22 19:51.

"Batteries included" is outdated - use the Github option to get the most recent software.

In short:

1. Download Tcl/Tk (8.4 version)

2. Download the software using the Zip button from the Github page: http://github.com/3dptv/3dptv or use the 

	git clone git@github.com:3dptv/3dptv.gi

3. Download compiler - Microsoft Visual C Express is the best choice on Windows, *nix has g++ included

4. Follow the compilation instructions to get the source code compiled together with the Tcl/Tk includes and libraries.
