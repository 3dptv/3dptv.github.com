"Batteries included" is outdated - use the SVN option

Submitted by alexl on Thu, 2011-09-22 19:51.

"Batteries included" is outdated - use the SVN option to get the most recent software.

In short:

1. Download Tcl/Tk (8.4 version)

2. Download SVN client (TortoiseSVN on Windows, SVNX on Mac OS X)

3. Download compiler - Microsoft Visual C Express is the best choice on Windows, *nix has g++ included

4. Using SVN client get the software by  svn checkout ... (see the instructions) TRUNK is recommended

5. Follow the compilation instructions to get the source code compiled together with the Tcl/Tk includes and libraries.
