Compilation with Microsoft Visual Studio


 

MVS environment

We recommend you use the Microsoft Visual Studio environment, we use Microsoft Visual C++ Express.  http://www.microsoft.com/express

Checkout the 3D-PTV project from Github:

http://github.com/3dptv/3dpv

or download the zip of the latest development version: https://github.com/3dptv/3dptv/zipball/master


Check that:

1. You install Tcl/Tk - note that you must install 8.4 branch (and not the newer ones as these were never tested). Download one from ActiveTcl website. We also recommend to install it in C:\Tcl (rather than C:\Program Files\Tcl)

2. download the latest version of 3D-PTV from here: https://github.com/3dptv/3dptv/zipball/master, extract it to C:\PTV

3. download the 3dptv.vcproj file (from here http://3dptv.github.com/system/files/3dptv.vcproj), put it in C:\PTV

4. Open your C++ Express -> Open Project -> Use C:/PTV/3dptv.vcproj file. It's an old file and probably needs to be converted, I do not know if it works or how it will proceed from here. If it works, so things might work just out of the box.

If it doesn't, I give you the major points to be done:

1. Open a new project in C++ Express, use Command Line project option (no windows graphics, not win32, nothing like that)
2. Add the header files (.h) and source files (.cpp) to the project via Add Source
3. Set up directories (see below the screenshot):
 
    a) add C:/Tcl/include (or something like that under C:/Tcl/) to the Include directories under the general settings or under Configuration Properties - > Linker -> Include Dir
    b) Set your working directory path correctly under Configuration Properties -> Debugging -> Working Directory (Working directory is the test directory in your folder)
 
    c) add C:/PTV/ptv.tcl file as an input to Configuration Properties -> Debugging -> Command Arguments
(steps 1,2, and 3a were supposed to be included in the 3dptv.vcproj file)


Set two paths in the project settings

+


Open the project and in the <Solution Explorer> at the top of the file tree you will see 3D-PTV. Right-click it and go to Configuration Porperties->Debugging->Command Arguments/Working Directory. Enter the paths according to your file structure that they point to ptv.tcl and the the folder where you have your folders res, img, parameters and cal.

And off you go...

+




Now if all is installed correctly, pressing Run (green arrow) button should compile and execute the software.


Run the code by clicking the green arrow. You should see the main window of the 3D-PTV GUI.
