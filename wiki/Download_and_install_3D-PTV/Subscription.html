Subscription

Subscription is very simple:

  1. Send your contact details (name, e-mail at least) to the ptv@ifu.baug.ethz.ch. Please attach your agreement to fullfil the conditions of the MIT-type license as it is provided below and in the attached file. If you want to contribute to the project, make sure you mention the github username. We will then add you to the member list of the 3D-PTV software developers.

You will be able to download the source code via git-checkout as described on http://github.comp/3dptv page. We would gratefully appreciate if our users also develop the software further and submit their patches, fixes and developments for the best of all the users. We encourage cooperation and collaboration on various aspects of the software and particle tracking experiments.
